# ThreadDeck 브랜드 가이드

> [English](BRAND.md)

## 포지셔닝

**이름:** ThreadDeck for Codex<br>
**짧은 이름:** ThreadDeck<br>
**한국어 태그라인:** Codex 작업을 한눈에.<br>
**영어 태그라인:** Codex tasks, at a glance.

ThreadDeck은 로컬 우선 물리 작업 대시보드입니다. 한눈에 보는 정보, 직접 조작, 개인정보 보호, Stream Deck Neo 형태를 중심으로 설명합니다. 공식 Codex 또는 Elgato 제품처럼 표현하지 않습니다.

## 독자적인 마크

기준 마크는 [`assets/plugin.svg`](../assets/plugin.svg)입니다. 네 개의 버튼 격자와 민트-파랑 작업 흐름을 C 형태로 결합했습니다. 제공된 자산을 다시 그리거나 늘리거나, 제공자 로고를 추가하거나, 내부에 글자를 넣지 않습니다.

권장 여백은 모든 방향에서 마크 너비의 4분의 1입니다. 작게 쓸 때는 마크만, 크게 쓸 때는 전체 프로젝트명과 함께 사용합니다.

## 색상

마크와 제품 UI는 렌더러의 기존 팔레트를 사용합니다.

| 역할 | 다크 모드 | 라이트 모드 |
|---|---:|---:|
| 바탕 | `#000000` | `#F9F9F9` |
| 기본 글자 | `#F2F6FA` | `#0D0D0D` |
| 보조 표면 | `#2F2F2F` | `#ECECEC` |
| 작업 초록 | `#10A37F` | `#10A37F` |
| 진행 파랑 | `#0285FF` | `#0285FF` |
| 경고 | `#F5A524` | `#AC4F23` |

보라색은 작업 중 추론 상태를 알리는 용도로만 사용하며 기본 브랜드 색이 아닙니다.

## 글꼴

`src/plugin.js`에 정의된 시스템 글꼴 묶음을 사용합니다. 한글은 Apple SD Gothic Neo로 대체됩니다. ThreadDeck은 OpenAI Sans, SF Pro 파일, 그 밖의 독점 글꼴 파일을 재배포하지 않습니다.

## 문서 이미지

- `pnpm run render-docs`가 만든 실제 렌더러 출력을 사용합니다.
- 개인정보 없는 제목과 안정적인 데모 상태를 사용합니다.
- 개요 이미지는 4 × 2 격자 전체를 보여줍니다.
- 기능 설명에는 정확한 개별 버튼 렌더를 사용하며 하드웨어를 흉내 내거나 직접 다시 그리지 않습니다.
- 저장소가 소유하지 않은 기본 서드파티 액션은 실제 아이콘을 복제하거나 런타임 출력인 것처럼 보이지 않게, 중립 안내임을 명시한 그림만 사용합니다.
- 화면 모드를 설명할 때는 라이트·다크를 모두 보여줍니다.
- 움직임이나 완료 동작을 보여줄 때는 `pnpm run render-animation`이 만드는 결정론적 GIF를 사용합니다.

## 이름과 출처 표시

처음에는 “ThreadDeck for Codex”, 이후에는 “ThreadDeck”을 사용합니다. 이 저장소 밖에서 다운로드를 제공할 때도 비공식 독립 프로젝트라는 고지를 포함해야 합니다. OpenAI, Codex, Elgato, Stream Deck 상표는 각 소유자에게 귀속됩니다. [고지](../NOTICE.ko.md)를 참고하세요.
