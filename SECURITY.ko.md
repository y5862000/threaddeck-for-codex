# 보안과 개인정보 보호

> [English](SECURITY.md)

ThreadDeck은 로컬에서만 작동합니다. 텔레메트리, 분석 도구, 업데이트 서버, 원격 API, Codex 작업 메타데이터를 업로드하는 코드는 없습니다.

## 로컬 접근 범위

- 플러그인은 사용자 작업 카드, 원격 턴 시간, 임시 사이드챗 수명 주기를 표시하기 위해 `~/.codex` 아래 Codex 상태와 Codex Desktop 로그를 읽습니다. 원격 카드는 Codex Desktop이 이미 로컬에 캐시한 메타데이터에서만 만들며 ThreadDeck이 원격 Mac과 직접 통신하지 않습니다. 내부 보조·검토 작업은 렌더링 전에 제외합니다. Codex 데이터베이스와 세션 파일에는 쓰지 않습니다.
- 선택 기능인 한도 링은 별도로 설치한 `codexbar` 명령을 실행하고 JSON 출력만 읽습니다.
- 키보드·미디어 동작은 작은 네이티브 헬퍼가 보내며, Stream Deck에 macOS 손쉬운 사용 권한이 필요합니다.
- 작업 전환과 작성창 컨트롤은 정확한 손쉬운 사용 포커스와 키보드 활성화만 사용합니다. Codex가 노출하면 헬퍼 내부에서 대상 작업 UUID를 먼저 일치시키고 개인정보 안전 제목 지문을 보조로 사용하며, 이미 알려진 같은 제목은 UUID 일치를 요구합니다. 헬퍼는 마우스 이동이나 좌표 클릭을 합성하지 않습니다.
- 같은 헬퍼가 화면에 보이는 Codex 대기 동작 버튼을 셉니다. 현재 창 제목과 버튼 라벨의 FNV-1a 지문 및 개수만 출력하며, 대기 메시지 원문은 무시되고 헬퍼 밖으로 나오지 않습니다.
- 포커스된 작업이 정확히 일치할 때는 고정된 추론 단계 값(`none`~`ultra`) 하나만 추가로 반환할 수 있습니다. 임의의 손쉬운 사용 텍스트는 헬퍼 밖으로 나오지 않습니다.
- 작업 버튼 음성 입력은 포커스된 편집 영역의 길이·해시 메타데이터만 비교해 받아쓰기 완료 여부를 판단합니다. 받아쓰기 원문은 플러그인으로 반환되지 않습니다.
- 음성 입력 전에 헬퍼는 앱 허용 목록 없이 활성 Core Audio 프로세스의 화면 앱을 찾고 의미가 확인된 일시정지 컨트롤만 사용하며, 필요할 때만 macOS 정상 미디어 명령으로 폴백합니다. 10분짜리 임시 임대 정보에는 멈춘 앱의 번들 식별자만 저장해 마지막 녹음 소유자가 해당 앱만 재개합니다. PID·제목·URL·미디어 문구는 저장하지 않고 프로세스에 신호를 보내거나 강제로 멈추지 않습니다. 종료 처리도 ThreadDeck이 일시정지를 소유한 경우에만 미디어를 복원합니다.
- ThreadDeck은 크기가 제한된 로컬 진단 기록을 `~/Library/Logs/ThreadDeck/runtime.jsonl` 또는 `THREADDECK_TRACE_PATH`에 씁니다. 256 KiB에서 비우며 시각, 이벤트·단계 이름, 슬롯 번호, 불리언, 단순 결과, 경과 밀리초만 포함합니다. 작업 제목·ID, 받아쓰기·대기 원문, 원격 호스트명, 손쉬운 사용 원문, 인증 정보는 이 기록이 받거나 저장하지 않습니다.

Codex 파일에 대한 접근이 읽기 전용이라는 말이 하드웨어가 수동 표시만 한다는 뜻은 아닙니다. 사용자가 버튼을 누르면 ThreadDeck은 의도대로 Codex UI를 열고, 받아쓰기를 시작·종료하고, 화면 작성창 컨트롤을 통해 받아쓴 메시지를 제출할 수 있습니다.

플러그인은 작업 제목을 표시하므로 Stream Deck을 볼 수 있는 사람에게 민감한 프로젝트명이 노출될 수 있습니다. 이슈 화면에는 실제 비공개 제목을 사용하지 마세요.

## 지원 버전

보안 수정은 최신 공개 베타를 대상으로 합니다. 신고 전에 최신 릴리스에서 재현되는지 확인해주세요.

## 취약점 비공개 신고

[GitHub Private Vulnerability Reporting](https://github.com/y5862000/threaddeck-for-codex/security/advisories/new)을 이용해주세요. 취약점은 공개 이슈로 올리지 마세요.

다음 자료는 절대 업로드하지 마세요.

- `~/.codex` 데이터베이스 또는 세션 파일
- API 키, 토큰, 쿠키, 인증 헤더
- Stream Deck 기기 일련번호 또는 하드웨어 식별자가 남은 내보낸 프로필
- 비공개 작업 제목이나 고객 정보가 포함된 화면

보안 외 질문은 [지원 안내](SUPPORT.ko.md)를 따릅니다.
